# Padrões Visuais Proibidos

Evite estes padrões:

## Visual amador

- Cards com cores aleatórias
- Gradientes fortes em todos os lugares
- Ícones enormes sem necessidade
- Botões com sombras exageradas
- Bordas muito grossas
- Layout muito apertado
- Falta de alinhamento entre títulos, cards e gráficos

## Dashboard ruim

- Muitos KPIs sem prioridade
- Gráficos antes das ações importantes
- Informações críticas escondidas no final da página
- Cards que só enfeitam e não ajudam na decisão
- Exibir dados estáticos fingindo que são dinâmicos
- Misturar financeiro, agenda e gestão sem separação clara

## Código ruim de UI

- Criar muitos componentes duplicados
- Colocar lógica complexa dentro do JSX
- Usar estilos inline sem necessidade
- Criar classes Tailwind gigantes e repetidas
- Quebrar responsividade existente
- Alterar dados dinâmicos sem entender de onde vêm
