# Direção Visual do Projeto

O projeto deve seguir uma estética moderna, profissional, minimalista e premium, com foco em clareza, hierarquia visual e experiência real do usuário.

## Estilo desejado

- Interface limpa, moderna e sofisticada
- Layout com bastante respiro visual
- Cards bem organizados, sem excesso de bordas
- Tipografia forte, com hierarquia clara
- Uso moderado de sombras suaves
- Cantos arredondados consistentes
- Cores neutras como base
- Cor principal usada apenas para ações, destaques e estados importantes
- Visual mais próximo de produtos SaaS profissionais do que de template genérico

## Sensação que a interface deve passar

- Confiança
- Organização
- Produto real de mercado
- Simplicidade
- Controle
- Profissionalismo

## Evitar

- Gradientes exagerados
- Muitos cards coloridos
- Ícones grandes demais
- Sombras pesadas
- Bordas muito marcadas
- Layout com cara de template gratuito
- Informações sem hierarquia
- Espaçamentos inconsistentes
- Telas que parecem painel administrativo antigo
